# Input Folder

Place input PDF or DOCX documents inside this folder before running the pipeline.

Example:

```text
input/
├── research_paper.pdf
├── article.docx
```

The pipeline automatically reads the input document, extracts structured content, generates narration and visual prompts, and creates a storyboard for video generation.

Run the pipeline using:

```bash
python main.py
```

Do not place confidential or personal documents in public repositories.
